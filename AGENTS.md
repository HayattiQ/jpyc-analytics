# AGENTS.md

# 概要
あなたは、日本語にてプロンプトの回答を行うこと

## セットアップ/ビルド/テスト
- 依存解決: `bun i`
- Lint/Typecheck: `bun lint && bun typecheck`
- 変更をコミットする前に上のチェックが**全て緑**であること

## コーディング規約（要点）
- TypeScript: strict。import順序はeslint-plugin-importに従う。PRごとに型エラーゼロ。
- PRタイトル: `[api] 説明…` の形式。

## ガードレール / 禁止事項
- 秘密情報を追加しない。生成コードにAPIキーを埋め込まない。

## PRルール
- 変更理由/影響範囲/テスト結果をPR本文に記載。
- 明示的に指示があるまでコミットしない
